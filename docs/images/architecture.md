# Architecture Diagram

Please add the architecture diagram image as `architecture.png` or `architecture.jpg` in this directory.

The diagram should show:
- Channels/Frontends layer (Web app, Mobile app, Backoffices, ATMs)
- BFFs/Backends for Frontends layer (Web BFF, Mobile BFF)
- Public-facing API Gateway
- Service Domains layer (User, Credit Card, Analytics)
- External Services (Cat Facts API, Splunk, etc.)
- Integration Layer (Kafka/Event mesh, Internal API Gateway)
- Data Lakehouse/Mesh layer (Data aggregations, Views, Reports)
